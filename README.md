# Pixel Cake APP

一个基于 Kotlin 的 Android 应用，提供 AI 图像处理和相机功能。

## 功能特点

- ✨ 智能相机捕获
- 🎨 AI 图像处理和修图
- 📚 丰富的预设滤镜
- 💾 本地数据库存储
- 🔄 历史记录管理
- 🎯 自定义预设创建

## 系统要求

- Android 10 (API 29) 或更高版本
- 支持相机的设备
- 至少 2GB RAM
- 至少 500MB 存储空间

## 快速开始

### 使用 GitHub Actions 构建

1. 访问 [Actions 页面](../../actions)
2. 点击 "Build Android APK"
3. 点击 "Run workflow"
4. 等待构建完成
5. 下载 APK 文件
6. 安装到 Android 设备

### 本地构建

```bash
cd android_native_project
./gradlew assembleDebug
